因为同时使用 Window 和 macOS 设备，为了方便起见，约定：

| WIndows | Mac     |
| ------- | ------- |
| ctrl    | ctrl    |
| alt     | option  |
| win     | command |
| click   | click   |

目前主要使用到的 IDE 有：

- JetBrains 系列
- Visual Studio Code
- Zed

主流 IDE 的功能、快捷键设计思路大都相近，麻烦的是我会在 Windows、Mac 和 Linux 环境下使用它们，为了保证体验的同意，特在此处制定一些使用较为频繁的快捷键，以便减少切换设备的磨合成本。
